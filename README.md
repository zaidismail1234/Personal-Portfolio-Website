# 🌐 Personal Portfolio Website  

A modern, responsive, and interactive *personal portfolio website* built using *HTML, CSS, and JavaScript*.  
It features *dual theme support (dark & light)*, smooth transitions, responsive design, and a professional UI/UX.  

---

## 🚀 Features
- 🎨 *Light & Dark theme* with toggle support  
- 📱 *Responsive design* for all devices  
- ⚡ Smooth animations & transitions  
- 🛠 Sections for *Skills, Projects, Education, Achievements*  
- 🖼 *Interactive project showcases* with modal previews  
- 📬 *Dynamic contact form* with validation  
- 📑 Downloadable *Resume*
